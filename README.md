# 📊 Análise de Combustíveis no Brasil

## Objetivo
Desenvolver um dashboard interativo para analisar os preços dos combustíveis no Brasil.

## Ferramentas
- Power BI
- Power Query
- DAX
- Excel

## Indicadores
- Preço médio por combustível
- Evolução dos preços
- Comparação entre estados
- Média por região
